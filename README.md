# Master of Logic Website

A modern website for Master of Logic - Software Engineering, AI & Automation organization.

## Tech Stack

- **React** - UI library
- **Vite** - Build tool and dev server
- **CSS** - Styling (separate CSS files for each component)

## Project Structure

```
├── src/
│   ├── components/
│   │   ├── Nav.js          # Reusable navigation component
│   │   ├── Nav.css
│   │   ├── Footer.js       # Reusable footer component
│   │   └── Footer.css
│   ├── pages/
│   │   ├── Home.js         # Home page component
│   │   └── Home.css
│   ├── App.js              # Main app component
│   ├── App.css             # Global styles
│   └── main.js             # Entry point
├── index.html
├── package.json
└── vite.config.js
```

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

4. Preview production build:
```bash
npm run preview
```

## Features

- ✅ Reusable Nav component with responsive design
- ✅ Reusable Footer component
- ✅ Modern hero section with gradient background
- ✅ Responsive design for mobile and desktop
- ✅ Separate CSS files for each component
